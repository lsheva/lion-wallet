Lion Wallet for Chrome

This installer places LionWallet.app in /Applications and configures the native messaging host for Chrome.

Install the Chrome Extension

  1. Locate the extension: /Applications/LionWallet.app/Contents/Resources/lion-wallet-chrome.zip
  2. Double-click the zip to extract it
  3. Open Chrome and go to chrome://extensions
  4. Enable Developer mode (toggle in the top-right corner)
  5. Click Load unpacked and select the extracted folder
Note: Restart Chrome after installation for the native messaging host to take effect.
